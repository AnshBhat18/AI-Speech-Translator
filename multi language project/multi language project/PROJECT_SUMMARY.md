# Project Summary

AI Speech Translator Pro is an intelligent multilingual translation platform that converts speech or text into different languages in real time.

Workflow:
1. User enters text or records speech.
2. System detects the input language automatically.
3. Text is translated into the selected target language.
4. Audio output is generated using Text-to-Speech.
5. Translation history is saved and visualized.

Future Scope:
- OCR translation from images
- Chat-based translator assistant
- Offline translation models
- User login and cloud history synchronization
