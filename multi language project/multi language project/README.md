# AI Speech Translator Pro

An upgraded Speech-to-Text and Translation AI application with a modern frontend.

## Features
- 🎤 Speech-to-Text Translation
- ✍️ Text Translation Mode
- 🌐 100+ Language Support
- 🔍 Automatic Language Detection
- 🔊 Text-to-Speech Output
- 📝 Translation History
- 📂 File Translation Support
- 📊 Translation Analytics Dashboard
- ⭐ Favorite Translations
- 🌙 Modern UI (Dark Mode Ready)

## Tech Stack
Python, Streamlit, SpeechRecognition, Deep Translator, Pyttsx3, LangDetect.

## Run
pip install -r requirements.txt
streamlit run app.py
